### 1) 干马天尼 Dry Martini（Gin）

* **配方**：金酒 60ml + 干味美思 10ml（想更干就 5ml）
* **做法**：加冰搅拌 20–30 秒，滤入冰镇鸡尾酒杯
* **点睛**：橙味苦精 1 dash（可选但很加分）；柠檬皮轻拧一下就够

### 2) 内格罗尼 Negroni（Gin + Campari）

* **配方**：金酒 30ml + 金巴利 30ml + 红味美思 30ml
* **做法**：加冰搅拌，倒入岩石杯加大冰
* **点睛**：橙皮一拧（不需要橙片/花）

### 3) 美利坚诺 Americano（Campari + 甜味美思）

* **配方**：金巴利 30ml + 红味美思 30ml + 苏打水补满
* **做法**：杯中加冰直调
* **口感**：比 Negroni 更轻盈、适合开场不醉人

### 4) 波本并非必须：布勒瓦迪耶 Boulevardier（Whiskey + Campari）

* **配方**：威士忌 45ml + 金巴利 30ml + 红味美思 30ml
* **做法**：搅拌，岩石杯大冰
* **推荐基酒**：你那支“**经典独奏葡萄酒桶**”很适合，果干/酒桶感会被 Campari 放大得很漂亮

### 5) 曼哈顿 Manhattan（Whiskey + 甜味美思）

* **配方**：威士忌 60ml + 红味美思 30ml + 橙味苦精 2 dash
* **做法**：搅拌，滤入鸡尾酒杯
* **推荐基酒**：**尊美醇**（柔顺）或 **葡萄酒桶威士忌**（更华丽）

### 6) 威士忌酸 Whiskey Sour（Whiskey）

* **配方**：威士忌 60ml + 柠檬汁 25ml + 单糖浆 15–20ml
* **做法**：摇壶加冰摇匀，滤入杯
* **精品升级（强烈推荐）**：加蛋白 20ml（先干摇再加冰摇），口感会变丝绒

### 7) 代基里 Daiquiri（白朗姆）

* **配方**：白朗姆 60ml +（最好青柠汁）25ml + 单糖浆 15ml
* **做法**：加冰猛摇，滤入鸡尾酒杯
* **推荐基酒**：蔗园三星更干净；哈瓦那三年更圆润

### 8) 莫吉托 Mojito（白朗姆）

* **配方**：白朗姆 50ml + 青柠 25ml + 糖浆 15ml + 薄荷 + 苏打补满
* **做法**：轻压薄荷（别捣烂），杯中加冰直调
* **关键**：这杯不靠花哨，靠“清爽、香气、气泡感”做高级

### 9) 自由古巴 Cuba Libre（朗姆 + 可乐）

* **配方**：哈瓦那三年 50ml + 青柠 10ml + 可乐补满
* **做法**：杯中加冰直调
* **小技巧**：青柠“少量但必须”，不然会甜腻

### 10) 玛格丽特 Margarita（银龙舌兰 + 君度）

* **配方**：银龙舌兰 50ml + 君度 20ml +（最好青柠汁）25ml
* **做法**：加冰猛摇，杯口可轻薄盐边（别厚盐）
* **备注**：用柠檬也能做，但青柠更“脆”和更像经典

### 1) Contessa（“更轻、更干净”的 Negroni 变体）

* **配方**：金酒 30ml + 阿佩罗 30ml + 干味美思 30ml
* **做法**：搅拌，岩石杯大冰
* **风格**：苦甜更柔、香气更高，非常适合跨年长聊

### 2) 50/50 Martini（更适口、更高级的马天尼）

* **配方**：金酒 45ml + 干味美思 45ml + 橙味苦精 1 dash
* **做法**：搅拌，鸡尾酒杯
* **推荐基酒**：季之美/日本手工金——草本会被“放大但不刺”

### 3) 冰冷竹林感：Martini 加橙苦精（香气更立体）

* **配方**：金酒 60ml + 干味美思 10ml + 橙味苦精 2 dash
* **做法**：搅拌
* **为什么好喝**：橙苦精把“干、冷、利落”变成“干、冷、但有层次”

### 4) El Presidente（朗姆的“优雅派对酒”）

* **配方**：哈瓦那三年 45ml + 干味美思 20ml + 君度 10ml + 石榴糖浆 1 barspoon（约 5ml）
* **做法**：搅拌，鸡尾酒杯
* **风格**：圆润、克制的甜、带一点微妙果香——很“精品酒吧”

### 5) Rosita（龙舌兰 × Campari × 双味美思）

* **配方**：银龙舌兰 45ml + 金巴利 15ml + 红味美思 15ml + 干味美思 15ml + 橙味苦精 2 dash
* **做法**：搅拌，岩石杯大冰
* **风格**：比 Negroni 更复杂、更干、更耐喝

### 6) “黑玛格” Margarita Negra（龙舌兰 + Amaro 的苦甜深度）

* **配方**：银龙舌兰 45ml + 君度 15ml + Amaro 15ml + 青柠 25ml（或柠檬 20ml）
* **做法**：摇匀，岩石杯加冰
* **风格**：不甜腻，苦甜拉开层次，适合喜欢“成熟口感”的人

### 7) Kingston-ish Negroni（用朗姆做 Negroni 的热带苦甜）

* **配方**：哈瓦那三年 30ml + 金巴利 30ml + 红味美思 30ml
* **做法**：搅拌，岩石杯大冰
* **风格**：朗姆带来的甘蔗/香草感会让苦甜更“圆”

### 8) 葡萄酒桶曼哈顿 Vin-Cask Manhattan（你那支过桶威士忌的主场）

* **配方**：葡萄酒桶威士忌 50ml + 红味美思 25ml + 干味美思 5ml + 橙味苦精 2 dash
* **做法**：搅拌，鸡尾酒杯
* **风格**：果干、木质、香料，层次像“成熟红酒的影子”

### 9) 海风烟熏高球 Smoky Highball（Talisker 10）

* **配方**：泰斯卡 10 年 40ml + 苏打水补满 +（可选）金巴利 5ml 点苦
* **做法**：杯中加大冰直调
* **风格**：干净、咸鲜烟熏、很“高级但不复杂”

### 10) 爱尔兰橙香酸 Irish Orange Sour（柔顺但很有香气）

* **配方**：尊美醇 55ml + 柠檬汁 25ml + 君度 10ml + 单糖浆 10–15ml + 橙味苦精 1 dash
* **做法**：摇匀（加蛋白更丝滑）
* **风格**：比普通 Whiskey Sour 更“香”、更细腻
